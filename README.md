#Java Quiz Project

##  Project Description

This is a console-based Java Quiz programthat asks users a series of questions and calculates a score based on their answers. The program is beginner-friendly and demonstrates basic Java concepts such as variables, conditionals, and user input.

The quiz consists of 4 multiple-choice questions. Each correct answer adds points to the userâ€™s score, and at the end, the program provides feedback based on performance
# Features

* 4 multiple-choice questions
* Scoring system: 5 points per correct answer
* Personalized feedback based on total score
* User input handled via Scanner
* Beginner-friendly and interactive
##  Technologies Used

* Java
* Scanner class
* if-else statements
* String comparison
* Console input/output
## How to Run

1. Save the file as:

   
   Quiz.java
  
2. Compile the program:

   bash
   javac Quiz.java
  
3. Run the program:

   bash
   java Quiz
